Welcome to the High Harmonic Generation Learning Tool!

Developed by Kristy Hudgins, Anand Hari Natarajan, and Nikolay Golubev.

***Please follow the instructions below to start using the learning tool!***

1. Use the phase slider to manually set phase value or input it in the phase window.

2. Click the "Ionize" button to start the simulation for the ionization phase you set!

3. Move the phase slider to look at the electron trajectories.

4. Click the "Ionize" button repeatedly to produce multiple trajectories and compare them.

5. Use the "Clear" button to clear the graphs.

6. Use the "Multiple Trajectories" button to generate a set of trajectories for various preset ionization phases.

6. The "Data" box tells the important information regarding the trajectories.
